              四晶片板雷射打標系統 - 快速開始指南
═══════════════════════════════════════════════════════════════

【第一步：編譯專案】
────────────────────────────────────────────────────────────
1. 使用 Visual Studio 開啟專案：
   C:\Users\AlexYang_VW\source\repos\MMProject\MMProjectMulti\WindowsFormsApp1\WindowsFormsApp1.sln

2. 在 Visual Studio 中：
   - 選擇平台：x64（非常重要！）
   - 選擇配置：Release 或 Debug
   - 點擊「建置」→「建置方案」

3. 編譯完成後，DLL 檔案會自動複製到輸出目錄

【第二步：執行應用程式】
────────────────────────────────────────────────────────────
1. 執行檔位置：
   bin\x64\Release\WindowsFormsApp1.exe
   或
   bin\x64\Debug\WindowsFormsApp1.exe

2. 雙擊執行程式

【第三步：使用應用程式】
────────────────────────────────────────────────────────────

▶ 1. 初始化晶片板
   - 點擊「初始化」按鈕
   - 等待提示「四個晶片板初始化完成！」

▶ 2. 設定線段參數
   在右側「線段參數」區域輸入：
   - X1: 起點 X 座標（例如：0）
   - Y1: 起點 Y 座標（例如：0）
   - X2: 終點 X 座標（例如：50）
   - Y2: 終點 Y 座標（例如：50）
   - 選擇板：選擇 1、2、3 或 4 號晶片板

▶ 3. 繪製線段
   - 點擊「繪製線段」按鈕
   - 線段會顯示在選定的晶片板預覽區域

▶ 4. 開始雷射打標
   - 確認線段位置無誤
   - 點擊「開始雷射」按鈕
   - 系統開始執行雷射打標

▶ 5. 停止操作（如有需要）
   - 點擊「停止」按鈕可緊急停止

【操作範例】
────────────────────────────────────────────────────────────
範例 1：在晶片板 1 上畫一條對角線
  X1 = 0,  Y1 = 0
  X2 = 50, Y2 = 50
  選擇板 = 晶片板 1

範例 2：在晶片板 3 上畫一條水平線
  X1 = 10, Y1 = 25
  X2 = 90, Y2 = 25
  選擇板 = 晶片板 3

範例 3：在晶片板 2 上畫一條垂直線
  X1 = 50, Y1 = 10
  X2 = 50, Y2 = 90
  選擇板 = 晶片板 2

【重要提示】
────────────────────────────────────────────────────────────
✓ 必須先點擊「初始化」才能使用其他功能
✓ 座標單位為毫米（mm）
✓ 每個晶片板可以獨立操作
✓ 建議在真實雷射前先在預覽區域確認圖形正確
✓ 使用雷射設備時請注意安全

【故障排除】
────────────────────────────────────────────────────────────
問題：提示「請先初始化！」
解決：點擊「初始化」按鈕完成系統初始化

問題：編譯錯誤「找不到 DLL」
解決：
  1. 確認 MarkingMate 已安裝在 C:\Program Files (x86)\MarkingMate
  2. 執行 CopyDlls.bat 手動複製 DLL
  3. 檢查平台設定是否為 x64

問題：初始化失敗
解決：
  1. 以系統管理員身分執行程式
  2. 檢查 MarkingMate 軟體是否正常運作
  3. 確認硬體連接正常
